# Student Result Prediction using Machine Learning

## Overview
This project predicts whether a student will **Pass** or **Fail** based on study hours and attendance using multiple Machine Learning algorithms. It compares different models and evaluates their performance using standard classification metrics.

## Features
- Data preprocessing
- Data visualization using Matplotlib
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest
- K-Means Clustering
- Train-Test Split
- Model evaluation and comparison

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree Classifier
- Random Forest Classifier
- K-Means Clustering

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score

## Dataset Features
- Hours_Study
- Attendance
- Result (Pass/Fail)

## Results
The project compares the performance of multiple machine learning models to identify the most suitable algorithm for predicting student results.
